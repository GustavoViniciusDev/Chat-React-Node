# Chat-React-Node
Chat usando react e node js


Comandos utilizados para a criação do projeto:
pasta client:

npm create vite@latest
  -react
  -javascript
  
npm install

npm i socket.io-client

pasta server:

npm init -y

npm i express socket.io

adicionar no package.json

no script:
"dev": "nodemon index.js",

npm i nodemon -D
